# 🚀 Abhishek Kumar — Personal Portfolio

A modern, responsive portfolio website built with React and Tailwind CSS.

## 🌐 Live Demo

[abhishek-portfolio.vercel.app](https://your-portfolio-link.vercel.app)

## ✨ Features

- Responsive design with dark theme
- Smooth animations and glassmorphism UI
- EmailJS contact form with auto-reply
- Project showcase with live links
- Skills marquee animation

## 🛠️ Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Email:** EmailJS
- **Deployment:** Vercel

## 📂 Sections

- Hero
- About
- Projects
- Experience
- Education
- Contact
